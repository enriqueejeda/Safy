# Safy
Safy | Un GPS peatonal inteligente desarrollado para mitigar el riesgo en entornos urbanos. Utiliza algoritmos de IA para analizar variables críticas (luminarias, comercios abiertos, reportes de incidencia) y trazar rutas seguras para mujeres, personas mayores y turistas. El objetivo: convertir el miedo en libertad de movimiento a través de datos.
